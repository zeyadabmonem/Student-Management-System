# Student Management System

A simple full-stack web application built with .NET and Vanilla JavaScript.

## Features
- **JWT Authentication**: Secure login system.
- **CRUD Operations**: Manage student records (Add, Edit, Delete, View).
- **SQLite Database**: Local data persistence.
- **Modern UI**: Responsive and clean design.

## Quick Start
1. Run the backend:
   ```bash
   dotnet run
   ```
2. Open the URL (e.g., `http://localhost:5003`) in your browser.
3. Login with:
   - **Username**: `admin`
   - **Password**: `password123`

## Project Structure
- `Controllers/`: API endpoints.
- `Models/`: Data models.
- `Data/`: Database context.
- `wwwroot/`: Frontend files (HTML, CSS, JS).
